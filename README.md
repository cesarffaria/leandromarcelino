# leandromarcelino

## must haves strapi 
yarn add @strapi/plugin-graphql

## must haves react
yarn add react-router-dom
yarn add apollo-boost @apollo/react-hooks graphql react-apollo
yarn add react-moment react-markdown moment
yarn add react-bootstrap bootstrap

## How to run

### strapi
`
npm run develop
`

### react
`
npm run start
`
